什么是 Git？

Git 是一个开源的分布式版本控制系统，用于敏捷和高效的项目处理。

版本控制

可以轻松将仓库中的内容恢复到之前的 commit。

原理：使用快照进行版本控制。

分布式

我的理解是，多个用户可以在不同的主机上，在不同的时间和空间内进行同一开发，而不会相互影响，最后统一提交。

基本原理

分为工作目录、暂存区和版本库。文件夹中有一个 .git 文件，包含暂存区和版本库，而 .git 之外的区域是工作目录。
